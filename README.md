# Тестовый проект: Hello World на C++

Этот проект создан в рамках тестового задания для изучения C++, CMake, Docker и Dev Containers.

## Требования
*   Docker
*   Visual Studio Code с расширениями Remote - Containers и C/C++

## Быстрый старт
После клонирования репозитория:
1.  Откройте папку проекта в VS Code.
2.  Нажмите `F1` и выполните команду **`Dev Containers: Reopen in Container`**.
3.  Дождитесь сборки контейнера (при первом запуске это займет несколько минут).
4.  Для сборки проекта нажмите `Ctrl+Shift+B`.
5.  Для запуска и отладки нажмите `F5`.

## Зависимости
Все зависимости (компилятор g++, CMake, GDB) автоматически устанавливаются при сборке Docker-образа.

## Структура проекта
test_task/
